## Personal dashboard with videos from Youtube

- Authorization through Magic Link
- Possibility to like/dislike videos
- Liked videos are saved in your list
- Rated videos are saved in a 'Watch it again' list
- DB - Hasura

## Techs used: 
- Next.js
- Typescript
- Magic link
- GraphQL
